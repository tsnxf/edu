# 教学参考 (Edu) - Code Wiki

## 1. 项目整体架构
本项目是一个纯前端的静态教学辅助 Web 应用。
- **技术栈**：纯 HTML5 + CSS3 + 原生 JavaScript (ES6+)，未引入任何前端框架（如 React/Vue）或构建工具（如 Webpack/Vite）。
- **运行环境**：现代浏览器。
- **数据存储**：由于没有后端服务，所有的用户注册和登录状态均通过浏览器的 `localStorage` 进行本地持久化存储。
- **页面路由**：多页面应用（MPA），通过原生的 `<a>` 标签在不同的 `.html` 页面间进行跳转。

## 2. 主要模块职责
项目按学科划分为不同的页面，并抽象出了公共的样式和鉴权逻辑。

- **首页模块 (`index.html`)**：网站的入口页面，当前作为占位页，后续可用于展示课程目录、系统公告和推荐内容。
- **数学模块 (`math.html`)**：实现“二次函数作图仿真”。支持探索模式、一般式、顶点式和焦点-准线式四种方程模式。通过 HTML5 Canvas 实时绘制抛物线、对称轴、顶点、焦点及准线。
- **物理模块 (`physics.html`)**：实现“理想气体定律”仿真。用户可以拖动滑块调节气体的物质的量 (n)、温度 (T) 和体积 (V)，程序实时计算压强并利用 Canvas 绘制出对应的 P-V 等温曲线。
- **化学模块 (`chemistry.html`)**：实现“酸碱混合 pH”仿真。模拟强酸和强碱的混合，通过调节酸碱的浓度和体积，实时计算混合溶液的 pH 值，并通过 DOM 操作动态改变烧杯中液体的颜色和高度。
- **鉴权模块 (`auth.js`)**：处理全局的用户注册、登录和登出逻辑，控制页面顶部导航栏的用户状态展示和弹窗交互。
- **公共样式 (`common.css`)**：定义了全站的 CSS 变量（主题色）、基础重置样式、顶部导航栏、学科切换 Tab 以及鉴权弹窗的通用样式。

## 3. 关键类与函数说明

### 3.1 鉴权模块 (`auth.js`)
- `initAuthUI()`: 初始化鉴权 UI 的核心函数，在各个页面底部被调用。负责绑定 DOM 元素的事件监听器。
- `getUsers()` / `setUsers(users)`: 与 `localStorage` 交互，获取或保存系统中所有注册用户的数据列表。
- `setCurrentUser(username)` / `getCurrentUser()`: 设置或获取当前登录用户的状态。
- `updateAuthView()`: 根据当前的登录状态，更新顶部导航栏（显示/隐藏登录注册按钮，显示会员名称和退出按钮）。
- `openAuthModal(mode)` / `closeAuthModal()`: 控制登录/注册弹窗的显示与隐藏，`mode` 参数区分当前是登录还是注册。
- `submitAuth()`: 处理弹窗表单的提交，进行表单校验、用户查重（注册）或密码比对（登录）。

### 3.2 数学模块 (`math.html`)
- `yVal(x)`: 根据当前选中的方程模式及对应的参数（a, b, c, h, k, p），计算并返回给定的 x 对应的 y 值。
- `coeffs()`: 无论当前是什么模式，都将其参数转换为一般式 $y = ax^2 + bx + c$ 的系数 $a, b, c$。
- `features()`: 计算二次函数的关键数学特征，包括对称轴、顶点坐标、实数根、y轴截距、焦点和准线。
- `equation()`: 根据当前模式，动态拼接并返回格式化后的数学方程字符串。
- `drawGrid()` / `drawCurve(fn, color)` / `point(x, y, c)`: 封装的 Canvas 绘图基础函数，用于绘制坐标系网格、函数曲线和关键点。
- `draw()`: 核心渲染函数，负责清空画布并调用各基础绘图函数，最终在界面上呈现二次函数图像及数学特征。

### 3.3 物理模块 (`physics.html`)
- `calcPhysics()`: 核心物理计算逻辑。利用理想气体状态方程 $PV = nRT$，根据当前的 $n, T, V$ 计算出压强（单位：kPa 和 atm）。
- `renderChart()`: 使用 HTML5 Canvas 绘制等温条件下的 P-V（压强-体积）反比例曲线，并在曲线上标出当前状态所在的点。
- `renderPhysics()`: 汇总函数，更新页面上的数值显示，并触发图表的重绘。

### 3.4 化学模块 (`chemistry.html`)
- `calcChemistry()`: 核心化学计算逻辑。计算酸和碱混合后的 $H^+$ 或 $OH^-$ 的过量摩尔数，并得出最终溶液的 pH 值以及酸碱性质。
- `phToColor(ph)`: 视觉映射函数。将 0-14 的 pH 值映射为 HSL 颜色值（酸性偏红，中性偏绿/黄，碱性偏蓝）。
- `renderChemistry()`: 更新页面数值，并动态设置 CSS 属性来改变“烧杯”中液体 div 的高度（基于总体积）和背景颜色（基于 pH 值）。

## 4. 依赖关系
- 本项目**无任何外部依赖**。
- 不依赖 npm 包、第三方 JS 库（如 jQuery, D3, ECharts）或 CSS 框架（如 Bootstrap, Tailwind）。
- 完全依赖现代浏览器内置的 API（ES6, HTML5 Canvas, LocalStorage）。

## 5. 项目运行方式
本项目是纯静态文件，运行方式极其简单：

**方法一：直接双击运行（推荐）**
1. 使用系统的文件管理器打开项目目录。
2. 双击 `index.html` 文件，即可在默认浏览器中预览并使用该系统。

**方法二：使用本地开发服务器（适合开发调试）**
1. 如果使用了 VS Code，可以安装 `Live Server` 插件，在 `index.html` 右键选择 "Open with Live Server"。
2. 或在项目根目录下使用 Python 启动简单的服务器：
   ```bash
   python3 -m http.server 8000
   ```
   然后在浏览器中访问 `http://localhost:8000`。
3. 或使用 Node.js 的 http-server：
   ```bash
   npx http-server .
   ```
